# Labour-Chowk-Website-
A responsive Labour Chowk frontend website developed using HTML, CSS, and Bootstrap. It includes Home, About Us, Contact Us, Workers, Jobs, Projects, Collaboration and Join Us pages to connect workers, employers, and contractors through an easy-to-use interface. 
## Features
- Home Page
- About Us Page
- Contact Us Page
- Workers Page
- Jobs Page
- Projects Page
- Collaboration Page
- Join Us Form
- Responsive Design
## Technologies Used 
- HTML5
- CSS3
- Bootstrap
## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. open 'project2.html' in any web browser.
